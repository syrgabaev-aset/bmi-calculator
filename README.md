1. Introduction

This assignment is focused on learning the basics of backend web development using Node.js and Express.
The goal of the task is to build a simple BMI (Body Mass Index) calculator that works through HTTP requests and demonstrates routing, request handling, and server-side logic.

2. Project Description

The BMI Calculator allows a user to enter their weight and height using a web form.
After submitting the form, the server calculates the BMI value and displays the result along with a health category such as underweight, normal weight, overweight, or obese.

3. Technologies Used

Node.js

Express.js

HTML

npm

4. Server Implementation

The backend server is built using Node.js and Express.
Routing is implemented to handle user requests and calculate BMI values on the server side.
